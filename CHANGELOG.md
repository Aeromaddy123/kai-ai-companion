# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Voice interaction with wake word detection
- Vision capabilities with LLaVA
- Persistent memory with ChromaDB
- Offline STT with Vosk
- Text-to-Speech with Piper
- REST API server with FastAPI
- Raspberry Pi client application
- Comprehensive documentation
- Setup scripts for server and RPI
- GitHub Actions CI/CD

### Features
- "Hey AEROMADDY" wake word activation
- Voice-to-voice conversations
- Image description and vision Q&A
- Semantic memory search
- Context-aware responses
- Multiple voice options

### Technologies
- Qwen 2.5 8B language model
- LLaVA 1.6 vision model
- Ollama runtime
- Raspberry Pi 4/5 support
- FastAPI backend
- ChromaDB vector database
